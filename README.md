## Рекомендации по формату и оптимизации

1. Рекомендуемый формат для всех крупных растровых изображений: **WebP**.
2. Для мобильных/десктопных фонов держать отдельные версии:
   - desktop: 1920px по ширине,
   - mobile: 1080px по ширине.
3. Для карточек тренеров/выпускников:
   - целевой размер: 800–1200px по ширине,
   - сжатие WebP с quality 70–82.
4. Для декоративных PNG-элементов (иконки/наклейки):
   - по возможности конвертировать в WebP с прозрачностью,
   - оставить PNG только если после конвертации заметны артефакты.
5. Добавлять `loading="lazy"` для всех не-критичных изображений (уже используется в ключевых местах).

## Какие файлы стоит конвертировать в WebP в первую очередь

- Все PNG из `assets/img/decor/sitebg/`
- Все PNG из `assets/img/coaches/` (новые фото тренеров)
